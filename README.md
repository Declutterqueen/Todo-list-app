# 📝 Advanced To-Do List App

A clean, responsive, and interactive **To-Do List web application** built with HTML, CSS, and JavaScript. Keep track of your tasks efficiently with features like task completion, deletion, dark mode, and real-time task counts.

🎯 **Live Demo:** [Try it here](https://declutterqueen.github.io/Todo-list-app/)

---

## Features

- Add, mark as done, and delete tasks.
- **Done button (`✔️`)** for marking tasks completed.
- **Delete button (`❌`)** to remove tasks.
- Real-time **task counters** for All, Pending, and Completed tasks.
- **Filter buttons** to view:  
  - All tasks  
  - Pending tasks  
  - Completed tasks
- **Dark mode** toggle for eye-friendly usage.
- **Responsive design** – works perfectly on mobile, tablet, and desktop.
- Persist tasks in **localStorage** so they remain after page reload.

## Installation

1. Clone the repository:

```bash
git clone https://github.com/Declutterqueen/Todo-list-app.git
```

2. Open `index.html` in your browser or host it using GitHub Pages.

---

## Usage

1. Enter a task in the input box.
2. Click **Add Task** or press **Enter** to add it to the list.
3. Click the **✔️ button** to mark a task as completed.
4. Click the **❌ button** to delete a task.
5. Use the **filter buttons** to view All, Pending, or Completed tasks.
6. Click **🌓 Toggle Dark Mode** to switch themes.
